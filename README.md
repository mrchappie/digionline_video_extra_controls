# DigiOnline Video Player Extra Controls

## Adds extra controls to the video player on DigiOnline.ro

## How to install

1. Clone or download the repository files to your computer. The files needed for this extension are: `manifest.json`, `content.js`
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the folder where the `manifest.json` file is located

## How to use

1. Open a video on DigiOnline.ro
2. The extra controls will be added to the video player, o the bottom right side of the video, after time progress
3. You can click on the +5s button to go 5 seconds forward in the video, the -5s button to go 5 seconds backward in the video or use right / left arrow keys
4. You can use the slider to set jump interval or use CTRL + right / left arrow keys to set the jump interval

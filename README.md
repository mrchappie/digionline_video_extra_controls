# DigiOnline Video Player Extra Controls

## Adds extra controls to the video player on DigiOnline.ro

## How to use

1. Clone or download the repository files to your computer. The files needed for this extension are: `manifest.json`, `content.js`
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the folder where the `manifest.json` file is located
